## 📝 Descripción

**UamoPS4** es un emulador de PlayStation 4 en etapa de desarrollo, diseñado específicamente para dispositivos Android. Actualmente se encuentra en una fase **altamente experimental**, enfocado en la investigación y el desarrollo de la arquitectura del sistema.

> **⚠️ Nota Importante:** Al ser una versión de prueba preliminar, es normal experimentar cierres inesperados (crashes), rendimiento inestable y una compatibilidad limitada.
